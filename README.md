# 🛒 POS System (C# WinForms)

A complete **Point of Sale (POS) System** built using **C# WinForms** and **SQL Server**. This project is designed as a beginner-friendly desktop application that demonstrates database connectivity, CRUD operations, and a clean multi-form UI structure.

## 📋 Features

- 👤 **Customer Management** — Add, view, and manage customer records
- 📦 **Product Management** — Add and track products with pricing and stock
- 🏭 **Supplier Management** — Manage supplier details and inventory sources
- 🧾 **Billing System** — Generate bills and track order details
- 🔐 **Login System** — Basic authentication to access the system

## 🛠️ Built With

- **C# (.NET / WinForms)** — Desktop application framework
- **SQL Server** — Database for storing customers, products, suppliers, and orders
- **Visual Studio** — IDE used for development

## 🚀 Getting Started

### Prerequisites
- Visual Studio (2022 or later recommended)
- SQL Server / SQL Server Express

### Installation

1. Clone or download this repository
2. Extract the `POS2.zip` file
3. Open `POS2.sln` in Visual Studio
4. Restore the database:
   - Open SQL Server Management Studio (SSMS)
   - Run the `POSDb.sql` script, **or** restore from `POSDb.bak`
5. Update the connection string in `App.config` if needed
6. Build and run the project (F5)

### Default Login
> ⚠️ Please set up your own admin username and password after restoring the database for security purposes.
